## A faire.
Terminale:
- Vecteurs dans $\mathbb{R}^3$.
- Limites de fonctions $f: \mathbb{R} \to \mathbb{R}$.
- Convexité d'une fonction avec $f^{\prime\prime}$.
- Orthogonalité dans $\mathbb{R}^3$.
- Logarithme népérien: $\ln$.
- Equations différentielles $y^{\prime} = ay + b$.
- Integration: $\int_{a}^{b}{f(t)\,dt}$.
- Combinatoire.
- Trigonométrie: $\sin$ et $\cos$.
- Variables aléatoires.
- Loi des grands nombres.
- Nombres complexes $\mathbb{C}$ : $a + ib$ et $re^{i\theta}$.
- Arithmétique modulaire dans $\mathbb{Z}$ : $a \equiv b \; [m]$.
- PGCD.
- Graphes et matrices $\mathcal{M}_{m, n}(\mathbb{R})$.
